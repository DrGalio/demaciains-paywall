# ReviewLens 🔍

**AI-Powered Review Sentiment Analyzer — Free & Open Source**

Paste your product reviews. Get instant sentiment analysis, common complaints, feature requests, and actionable insights.

## What It Does

1. **Paste reviews** from Amazon, Etsy, Shopify, or any platform
2. **Get instant analysis:**
   - Positive/Negative/Neutral sentiment breakdown
   - Top praised features
   - Top complaints (ranked by frequency)
   - Theme extraction (quality, shipping, support, etc.)
   - Auto-generated action items with priority levels

## Why This Exists

24M+ e-commerce stores struggle with review analysis. Existing tools charge $50-200/month for what you can do in a browser.

ReviewLens runs entirely in your browser. No data sent to any server.

## Features

- Sentiment analysis (positive/negative/neutral)
- Theme extraction from review text
- Complaint frequency ranking
- Auto-generated action items
- Export to JSON or PDF
- 100% client-side (privacy-first)

## Quick Start

1. Open `index.html` in any browser
2. Paste your reviews (separated by `---` or blank lines)
3. Click "Analyze Reviews"

## Deploy

- **GitHub Pages:** Push → Settings → Pages → Deploy
- **Vercel:** Connect repo → Auto-deploy
- **Embedded:** Add to any website with an iframe
