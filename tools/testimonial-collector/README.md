# ⭐ SocialProof

**Collect & Display Testimonials — Free & Open Source**

A simple, beautiful tool for freelancers and small businesses to collect client testimonials and display them as social proof on their website.

## Why SocialProof Exists

Solo entrepreneurs and freelancers need testimonials but hate asking for them. Existing tools charge $10-30/month for what should be free.

SocialProof gives you:
- A shareable link for clients to submit testimonials
- A beautiful testimonial wall
- Embeddable widget for your website
- Star ratings and role/company info
- Stats dashboard

All data stored locally. No server. No subscription.

## Features

- ⭐ Star rating system
- 📝 Testimonial collection form
- 🖼️ Beautiful display wall
- 📋 Embeddable widget code
- 📊 Stats dashboard (count, avg rating, 5-star count)
- 💾 LocalStorage persistence
- 📤 Export testimonials as JSON

## Quick Start

1. Open `index.html` in any browser
2. Share the link with clients
3. Testimonials appear on your wall
4. Copy the embed code to your website

## Use Cases

- Freelancer portfolio sites
- Small business websites
- Agency landing pages
- Course creator testimonials
- Product reviews
