# Gym & Fitness Studio AI Agent Config Pack

A complete, ready-to-paste AI configuration that turns ChatGPT, Claude, or Gemini into your gym's virtual front desk — handling class schedules, membership questions, personal training info, workout guidance, nutrition basics, and member support 24/7.

---

## What's In This Pack

| File | Description |
|------|-------------|
| `system-prompt.md` | 3,000+ word AI personality & behavior configuration — covers class schedules, memberships, personal training, workout plans (beginner/intermediate/advanced), nutrition guidance, gym etiquette, injury modifications, equipment tips, and more. |
| `knowledge-base.md` | Customizable template with gym info, membership tiers & pricing, class schedule, personal training packages, facility amenities, 20+ FAQ pairs, and all policies (cancellation, freeze, guest passes, refunds, code of conduct). |
| `conversation-starters.md` | 10 high-value conversation starters covering new member welcome, class inquiries, personal training, weight loss, muscle building, membership comparison, injury modifications, nutrition, gym tours, and membership changes. |
| `setup-guide.md` | Step-by-step setup for ChatGPT Custom GPT, Claude Projects, and Gemini Gems. Includes ROI calculator, customization checklist, and best-practice tips. |
| `index.html` | Sales page for this product. |

## How It Works

1. **Customize** — Replace `[bracketed placeholders]` in `knowledge-base.md` with your gym's actual info
2. **Paste** — Drop the system prompt + knowledge base into ChatGPT, Claude, or Gemini
3. **Launch** — Your AI fitness assistant is live in 15 minutes, handling member questions 24/7
